# Maze-Escape
Triangle maze escape game</br>
MST의 성질을 활용하여 삼각형 모양 미로를 만들고, openframeworks 라이브러리를 사용하여 탈출 게임을 제작했습니다.
서강대학교의 컴퓨터공학설계 및 실험1 (CSE3013) 과목의 최종프로젝트로 제출했습니다.
</br></br>

## 🗂️ 파일 구조
```
main.cpp
│
├── utils.h
│   └── utils.cpp
│
└── matmul.h

```
